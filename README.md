Bluestock Mutual Fund Analytics Capstone
Project Overview
This project focuses on building a complete Mutual Fund Analytics Platform using Python, SQLite, and Power BI. The objective was to analyze mutual fund performance, investor behavior, portfolio diversification, and risk metrics while creating an interactive dashboard for decision-making.


Project Objectives
•	Build an automated ETL pipeline
•	Store and manage data using SQLite
•	Perform Exploratory Data Analysis (EDA)
•	Calculate performance and risk metrics
•	Develop advanced analytics models
•	Create an interactive Power BI dashboard
•	Generate fund recommendations
